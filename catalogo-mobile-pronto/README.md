# Catálogo Interativo Mobile

Projeto acadêmico desenvolvido em React Native com Expo para exibir produtos de uma loja online, separados por categorias masculinas e femininas.

## Tecnologias utilizadas

- React Native
- Expo
- Axios
- React Navigation
- DummyJSON API

## Funcionalidades

- Tela de login com validação de campos
- Armazenamento temporário do usuário com useState
- Listagem de produtos por categoria
- Categorias masculinas e femininas
- Consumo de API REST real com Axios
- Tela de detalhes do produto usando parâmetro de ID
- Logout com limpeza dos dados armazenados

## API utilizada

https://dummyjson.com/products/category/{categoria}

Categorias masculinas:
- mens-shirts
- mens-shoes
- mens-watches

Categorias femininas:
- womens-bags
- womens-dresses
- womens-jewellery
- womens-shoes
- womens-watches

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o projeto:

```bash
npx expo start
```

3. Para abrir no navegador:

```bash
npx expo start --web
```

ou aperte `w` no terminal do Expo.
